<h1>Vue3 + ElementPlus + Vite4后台管理模板</h1>

## 技术栈

- Vue3
- ElementPlus
- Vite

## 开发

- 依赖安装

```bash
pnpm install
```

- 启动

```bash
pnpm dev
```

- 打包

```bash
pnpm build
```
